# PRODIGY_ML_02: Customer Segmentation Using K-Means Clustering

This repository contains **Task 02** completed during my Machine Learning Internship at **Prodigy InfoTech**.

## 📋 Task Overview
Implement an unsupervised learning algorithm (K-Means Clustering) to segment retail store customers based on their purchasing behavior and demographic data.

## 📊 Dataset
The model utilizes the popular [Mall Customers Dataset](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python) from Kaggle, analyzing:
* **Annual Income (k$)**
* **Spending Score (1-100)**

## 🛠️ Implementation & Insights
* **Optimal Clusters ($K$):** Identified as **5** using the mathematical Elbow Method (Within-Cluster Sum of Squares evaluation).
* **Segmentation Profiles:** Successfully grouped consumer behaviors into 5 strategic business profiles: Sensible Spenders, Standard Shoppers, Target Consumers, Careless Spenders, and Frugal Profiles.
